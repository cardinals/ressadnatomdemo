WCMCHANNELSYN表：

描述：

存储频道的同步分发信息

结构：

列名 说明 默认值 允许空值 类型 长度 备注
CHANNELSYNID 编号 int
SRCCHANNEL 源栏目编号 int WCMCHANNEL.CHANNELID
TOCHANNEL 目标栏目编号 int WCMCHANNEL.CHANNELID
WHERESQL 检索条件 nvarchar 500
SDATE 分发有效时间(开始) datetime
EDATE 分发有效时间(结束) datetime
DOCSDATE 允许分发的文档创建时间(开始) datetime
DOCEDATE 允许分发的文档创建时间(结束) datetime
STATUSES 允许分发的文档状态 nvarchar 300
ATTRIBUTE 扩展属性 nvarchar 500

备注

  1. SDATE,ENDATE:分发在这个时间区间有效
  2. DOCSDATE,DOCEDATE:只有在这个时间区间内创建的文档才会分发
