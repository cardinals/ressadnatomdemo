WCMDOCUMENT表：

描述：

文档信息

结构：

列名 说明 默认值 允许空值 类型 长度 备注
DOCID 编号 int
DOCCHANNEL 所属栏目 int
DOCVERSION 版本号 0 int
DOCTYPE 类型 int 类型
DOCTITLE 文档标题 nvarchar 1000
DOCSOURCE 文档来源 0 int WCMSOURCE.SOURCEID
DOCSECURITY 安全级别 int WCMSECURITY.SECURITYID
DOCSTATUS 文档状态 int WCMSTATUS.STATUSID
DOCKIND 文档分类 int 分类
DOCCONTENT 纯文本内容 ntext
DOCHTMLCON HTML文本内容 ntext
DOCABSTRACT 内容摘要 nvarchar 2000
DOCKEYWORDS 关键词 nvarchar 200
DOCRELWORDS 相关词 nvarchar 200 相关词
DOCPEOPLE 涉及的人物 nvarchar 200 首页标题
DOCPLACE 涉及的地址 nvarchar 200
DOCAUTHOR 作者 nvarchar 200
DOCEDITOR 编辑 nvarchar 50
DOCAUDITOR 编审 nvarchar 50
DOCOUTUPID 输入编号 int 输入编号
DOCVALID 有效时间 datetime
DOCPUBURL 发布后的链接地址 nvarchar 400
DOCPUBTIME 发布时间 datetime
DOCRELTIME 撰写时间 datetime
CRUSER 对象创建者 nvarchar 50 WCMUSER.USERNAME
CRTIME 对象创建时间 datetime
DOCWORDSCOUNT 纯文本内容单词数 0 int
DOCPRO 文档属性 1 int
RIGHTDEFINED 独立的权限设置 0 smallint
TITLECOLOR 标题的颜色 varchar 20
TEMPLATEID 设置的模板编号 0 int
SCHEDULE 计划任务编号 0 int
DOCNO 其它编号 nvarchar 4000 其它编号
DOCFLAG 标记 0 int 标记
EDITOR 文档编辑器 nvarchar 100 保留
ATTRIBUTE 扩展属性 nvarchar 500
HITSCOUNT 点击数 0 int
DOCPUBHTMLCON 发布HTML内容 ntext
SUBDOCTITLE 副标题 nvarchar 500
ATTACHPIC 是否附图 smallint
DOCLINK 链接地址 nvarchar 100 链接地址
DOCFILENAME 外部文件名 nvarchar 50
DOCFROMVERSION 从哪个版本恢复 int
OPERTIME 删除时间 datetime
OPERUSER 删除操作用户 nvarchar 50
FLOWOPERATIONMARK --列说明-- nvarchar 200
FLOWPREOPERATIONMARK --列说明-- nvarchar 200
FLOWOPERATIONMASKENUM --列说明-- nvarchar 2000
DOCSOURCENAME 文档来源名称 nvarchar 100
DOCLINKTO --列说明-- nvarchar 300
DOCMIRRORTO --列说明-- nvarchar 300
RANDOMSERIAL 表单文档随机查询号 nvarchar 100
POSTUSER 表单文档提交用户 varchar 50

备注

  1. 类型：四个有效值[文本文档=10,HTML文档=20,链接文档=30,外部文件文档=40]
  2. 分类：在元数据选件中,该列值是所属视图编号
  3. 首页标题：普通文档用作首页标题,在图片库是图片涉及的相关人物
  4. 输入编号：一般可用于数据交换,记录外部的唯一编号
  5. 其它编号：在表单选件中,记录表单文档的唯一编号(如果有的话);场景式服务亦有用
  6. 标记：在表单选件中,记录的是所属栏目的表单编号(WCMINFOVIEW.INFOVIEWID)
  7. 在图片库中,记录各缩放图的文件名
