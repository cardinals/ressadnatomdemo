WCMAPPENDIX表：

描述：

存储文档附件的基本信息，建立系统附件对象和文件系统的文件关联

结构：

列名 说明 默认值 允许空值 类型 长度 备注
APPENDIXID 文档附件编号 int
APPDOCID 文档附件所属的文档编号 int WCMDOCUMENT.DOCID
APPFILE 对应的文件系统文件名 nvarchar 255
APPFILETYPE 文档附件的文件类型 -1 int
APPDESC 文档附件的描述 nvarchar 255
APPLINKALT 文档附件的页面ALT nvarchar 255
APPSERN 文档附件在文档中的序号 int
APPPROP 文档附件的属性 int
APPFLAG 文档附件的标记 smallint
APPTIME 文档附件的时间 datetime
APPAUTHOR 文档附件的作者 nvarchar 100
APPEDITOR 文档附件的编者 nvarchar 100
ATTRIBUTE 文档附件的扩展属性 nvarchar 500
USEDVERSIONS 文档附件的版本号 1 numeric
CRUSER 对象创建者 nvarchar 50 WCMUSER.USERNAME
CRTIME 对象创建时间 datetime
SRCFILE 文档附件的原文件名 nvarchar 200
FILEEXT 文档附件的扩展名 nvarchar 20
ISIMPORTTOIMAGELIB 附件是否来自图片库 0 int 图片库
RELATEDCOLUMN --列说明-- varchar 200

备注:

  1. 图片库：当且仅当有图片库选件时有效。值为1时表示是来自图片库
