WCMINFOVIEWGROUP表：

描述：

自定义表单一个视图内部的数据块，对应InfoPath的Section或者Reapeating Table

结构：

列名 说明 默认值 允许空值 类型 长度 备注
IVGROUPID 编号 int
INFOVIEWID 所属表单编号 int WCMINFOVIEW.INFOVIEWID
IVVIEWID 所属视图编号 0 int WCMINFOVIEWVIEW.IVVIEWID
GROUPNAME 分组名称 nvarchar 50
GROUPDESC 分组描述 nvarchar 200
PUBLICFILL 是否公开可填写 1 tinyint
PUBLICVIEW 是否公开可查看 1 tinyint
SPECIALRIGHT 是否特殊权限可设置 0 tinyint
