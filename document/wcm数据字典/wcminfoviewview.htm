WCMINFOVIEWVIEW表：

描述：

自定义表单:内部视图(多视图表单)

结构：

列名 说明 默认值 允许空值 类型 长度 备注
IVVIEWID 编号 int
INFOVIEWID 所属表单编号 int WCMINFOVIEW.INFOVIEWID
VIEWNAME 视图名称 nvarchar 50
VIEWDESC 视图描述说明 nvarchar 200
OLDXSLFILE 旧的xsl文件 nvarchar 50
NEWXSLFILE 新的xsl文件 nvarchar 50
VIEWWIDTH 视图宽度 nvarchar 16
PUBLICFILL 是否可公开填写 1 tinyint
PUBLICVIEW 是否可公开查看 1 tinyint
DEFAULTVIEW 是否为默认视图 1 tinyint
SPECIALRIGHT 特殊权限 0 tinyint
XSLPREFIX xsl表达式前缀 nvarchar 50
