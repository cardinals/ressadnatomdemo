WCMDBUPDATELOG表：

描述：

WCM系统数据库更新日志记录(保留)

结构：

列名 说明 默认值 允许空值 类型 长度 备注
DBUPDATELOGID 编号 int
LOGTITLE 日志标题 nvarchar 100
LOGMEMO 更新详细说明 nvarchar 2000
SUBMITOR 提交者 nvarchar 50
TABLENAME 更新修改的系统表名 nvarchar 50
LOGTYPE 类型 int
SQLSERVER MS SQLServer的脚本 ntext
ORACLESQL Oracle的脚本 ntext
DB2SQL DB2的脚本 ntext
SYSBASESQL Sybase的脚本 ntext
CRUSER 对象创建者 nvarchar 30 WCMUSER.USERNAME
CRTIME 对象创建时间 datetime
