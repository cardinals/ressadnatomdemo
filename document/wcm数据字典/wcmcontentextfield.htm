WCMCONTENTEXTFIELD表：

描述：

扩展字段使用关联信息
表示扩展字段被哪个栏目/站点使用

结构：

列名 说明 默认值 允许空值 类型 长度 备注
CONTENTEXTFIELDID 编号 int
OBJID 对象编号 int
OBJTYPE 对象类型 int
EXTFIELDID 扩展字段编号 int WCMEXTFIELD.EXTFIELDID
LOGICFIELDDESC 字段逻辑显示名称 nvarchar 200
CRUSER 对象创建者 nvarchar 30 WCMUSER.USERNAME
CRTIME 对象创建时间 datetime

备注

  1. OBJID,OBJTYPE共同确定字段与哪个对象关联
    对于类型,一般是三个有效值[栏目=101,站点=103,站点根=0]
