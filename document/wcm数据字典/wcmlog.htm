WCMLOG表：

描述：

操作日志信息

结构：

列名 说明 默认值 允许空值 类型 长度 备注
LOGID 编号 int
LOGTYPE 日志类型 int
LOGDESC 日志描述 nvarchar 3000
LOGOBJNAME 操作对象名称 nvarchar 3000
LOGOBJTYPE 操作对象类型 int
LOGOBJID 操作对象编号 int
LOGOPTYPE 操作类型 int
LOGOPARGS 操作参数 nvarchar 1000
LOGRESULT 操作结果 smallint
LOGUSER 操作用户 nvarchar 50 WCMUSER.USERNAME
LOGOPTIME 操作时间 datetime
ATTRIBUTE 扩展属性 nvarchar 1000
PARENTID 父日志编号 0 int
STIMEMILLIS 操作开始时间戳 0 numeric
ETIMEMILLIS 操作结束时间戳 numeric
EXECTIME 操作执行耗时 0 numeric
LOGUSERIP 操作用户的登录IP nvarchar 50
TOPID 顶层日志编号 0 int
SIGNINFO 签名信息 nvarchar 1000
