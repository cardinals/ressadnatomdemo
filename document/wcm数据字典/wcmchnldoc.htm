WCMCHNLDOC表：

描述：

栏目和文档之间的关联信息

结构：

列名 说明 默认值 允许空值 类型 长度 备注
RECID 编号 int
CHNLID 栏目编号 int WCMCHANNEL.CHANNELID
DOCID 文档编号 int WCMDOCUMENT.DOCID
DOCORDER 序号 0 int
DOCSTATUS 文档状态 0 int
CRUSER 对象创建者 'admin' nvarchar 100 WCMUSER.USERNAME
CRTIME 对象创建时间 datetime
DOCPUBTIME 文档发布时间 datetime
DOCPUBURL 文档发布后的链接地址 nvarchar 300
DOCORDERPRI 文档置顶序号 0 int
INVALIDTIME 文档置顶失效时间 datetime
OPERUSER 删除文档的操作用户 nvarchar 50 WCMUSER.USERNAME
OPERTIME 文档被删除的时间 datetime
MODAL 引用类型 1 int 引用类型
DOCRELTIME 文档撰写时间 datetime WCMDOCUMENT.DOCRELTIME
DOCCHANNEL 所属栏目 int WCMDOCUMENT.DOCCHANNEL
DOCFLAG 标记 int WCMDOCUMENT.DOCFLAG
DOCKIND 分类编号 0 int WCMDOCUMENT.DOCKIND

  1. 类型：三个有效值[实体=1,链接引用=2,镜像引用=3]
