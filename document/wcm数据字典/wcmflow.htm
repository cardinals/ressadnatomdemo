WCMFLOW表：

描述：

工作流信息

结构：

列名 说明 默认值 允许空值 类型 长度 备注
FLOWID 编号 int
FLOWNAME 名称 nvarchar 50
FLOWDESC 描述 nvarchar 200
SNAME 起始节点名 nvarchar 100
ENAME 结束节点名 nvarchar 100
ATTRIBUTE 扩展属性 nvarchar 500
CRUSER 对象创建者 nvarchar 50 WCMUSER.USERNAME
CRTIME 对象创建时间 datetime
SITEID 所属站点编号 int WCMWEBSITE.SITEID
OWNERTYPE 所属对象类型 int
OWNERID 所属对象编号 int
INFOVIEWID 关联表单编号 int WCMINFOVIEW.INFOVIEWID
OPERATIONMARKENUM 工作流文档操作标签枚举值 nvarchar 2000 WCMDOCUMENT.OPERATIONMARKENUM

备注

  1. OWNERTYPE,OWNERID共同确定工作所属对象(站点/栏目等),SITEID为保留字段
  2. 表单关联编号在使用表是选件时有效
