WCMEXTFIELD表：

描述：

系统扩展字段

结构：

列名 说明 默认值 允许空值 类型 长度 备注
EXTFIELDID 编号 int
TABLENAME 扩展字段所属表名 nvarchar 50 表名
FIELDNAME 名称 nvarchar 50
FIELDTYPE 类型 nvarchar 10
FIELDMAXLEN 最大长度 int
FIELDNULLABLE 是否允许空值 1 smallint
FIELDDEFAULT 默认值 nvarchar 50
ATTRIBUTE 扩展属性 nvarchar 500
CRUSER 对象创建者 nvarchar 50 WCMUSER.USERNAME
CRTIME 对象创建时间 datetime
FIELDSCALE 精度 0 int

备注

  1. 表名：默认扩展字段只作用于WCMDOCUMENT表.每个记录记录两次,另一条记录对应于WCMDOCBAK表
