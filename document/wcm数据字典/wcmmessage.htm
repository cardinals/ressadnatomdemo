WCMMESSAGE表：

描述：

在线短消息记录

结构：

列名 说明 默认值 允许空值 类型 长度 备注
MSGID 编号 int
FLAG 标记 smallint
TITLE 标题 nvarchar 200
MBODY 详细内容 nvarchar 1000
VALID 有效时间 datetime
ATTRIBUTE 扩展属性 nvarchar 500
CRUSER 对象创建者 nvarchar 50 WCMUSER.USERNAME
CRTIME 对象创建时间 datetime
MSGTYPE 类型 1 int
SENDTYPE 发送方式 1 nvarchar 50
DELETEDBYSENDER 是否被分发者删除 0 int
MSGRECEIVER 接收者 nvarchar 50
BIGMBODY 内容(当前消息内容很长时使用) text
