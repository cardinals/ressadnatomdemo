WCMWEBSITE表：

描述：

站点信息

结构：

列名 说明 默认值 允许空值 类型 长度 备注
SITEID 编号 int
SITENAME 名称(唯一标识) nvarchar 200
SITEDESC 描述说明 nvarchar 200
FTPSERVER 分发:FTP目标 nvarchar 200 分发
FTPUSER 分发:FTP用户 nvarchar 50
FTPPASSWORD 分发:FTP密码 nvarchar 50
FTPFOLDER 分发:FTP存放目录 nvarchar 200
DATAPATH 存放位置 nvarchar 200 发布设置
WEBHTTP HTTP域 nvarchar 200 发布设置
SITEPROP 站点属性 int
PARENTID 父站点编号 int
SITEORDER 序号 0 int
ATTRIBUTE 扩展属性 nvarchar 500
HOMETEMPLATEID 首页模板编号 int 模板
SITEORDERBY 排序方式 nvarchar 200
CRUSER 对象创建者 nvarchar 50 WCMUSER.USERNAME
CRTIME 对象创建时间 datetime
PUBSTATUSES 允许发布的文档状态 varchar 200 发布状态
SCHEDULE 计划任务编号 0 int 计划任务
STATUS 状态 0 int
PUBLISHPRO 发布属性 1 smallint 保留
OPERUSER 删除站点用户名 nvarchar 50 WCMUSER.USERNAME
OPERTIME 被删除的时间 datetime
SITETYPE 类型 0 int

备注

  1. 分发:相关字段保留,信息见WCMPUBLISHDISTRIBUTION表
  2. 发布设置:相关字段保留,信息见WCMFOLDERPUBLISHCONFIG表
  3. 模板:保留,见WCMTEMPLATEEMPLOY表
  4. 发布状态:保留,见WCMPUBSTATUSCONFIG表
  5. 计划任务:计划任务
