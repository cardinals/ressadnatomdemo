WCMSCHEDULE表：

描述：

系统计划任务信息

结构：

列名 说明 默认值 允许空值 类型 长度 备注
SCHID 编号 int
SCHNAME 名称 nvarchar 300
SCHDESC 描述说明 nvarchar 200
OPTYPE 任务调度类型 int WCMOPER.OPERID
OPARGS 调度参数 nvarchar 1000
SCHMODE 调度类型 smallint 类型
SDATE 任务开始时间 datetime
EDATE 任务结束时间 datetime
ETIME 任务执行时间 datetime
PARAM 任务间隔时间 int
ATTRIBUTE 扩展属性 nvarchar 1000
CRUSER 对象创建者 nvarchar 50 WCMUSER.USERNAME
CRTIME 对象创建时间 datetime
LASTEXETIME 任务上次执行时间 datetime
LASTEXERES 任务上次执行是否成功 smallint 保留
LASTTIMEUSED 执行耗时 0 int
SENDERTYPE 计划任务请求者类型 0 int
SENDERID 计划任务请求者编号 0 int
SCHSTATUS 当前状态 1 smallint 保留

备注

  1. 类型：共三种类型[只运动一次=3,每天运行多次=2,每天运行一次=1]
  2. SDATE,EDATE,PARAM只对每天运行多次有效
  3. ETIME对每天运行多次的计划无效
