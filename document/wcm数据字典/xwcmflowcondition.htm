XWCMFLOWCONDITION表：

描述：

工作流操作条件

结构：

列名 说明 默认值 允许空值 类型 长度 备注
FLOWCONDITIONID 编号 int
HANDLER 执行者 '0' nvarchar 100
PARAMNAME 参数名称 '0' nvarchar 50
PARAMVALUE 参数值 nvarchar 50
RELATIONOPERATOR 比较操作符 nvarchar 50 如大于,小于等
ISANDOPERATOR 是否为与操作 1 smallint
RULEID 规则编号 0 int XWCMFLOWRULE.FLOWRULEID
CONDITIONORDER 序号 0 int
CRUSER 对象创建者 nvarchar 30 WCMUSER.USERNAME
CRTIME 对象创建时间 datetime
UNITVALUE 参数值单位 nvarchar 20 如时间单位:分钟
