工作流连线(分支)的基本信息

结构：

列名 说明 默认值 允许空值 类型 长度 备注
BRANCHID 编号 int

NEXTNAME 下一节点名称 nvarchar 50
STATUS 状态 int

OPERATIONMARK 操作标签 nvarchar 200
OPERATIONMARKENABLE 操作标签是否有效 smallint
