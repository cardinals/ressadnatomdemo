WCMCHANNEL表：

描述：

存储频道(栏目)的基本信息

结构：

列名 说明 默认值 允许空值 类型 长度 备注
CHANNELID 编号 int
SITEID 所属站点编号 int WCMWEBSITE.SITEID
CHNLNAME 名称(唯一标识) nvarchar 200 名称
CHNLDESC 描述(显示名称) nvarchar 200
CHNLTABLE 数据检索表 nvarchar 200
CHNLQUERY 检索条件 nvarchar 500
CHNLPROP 属性 int 属性
PARENTID 父栏目编号 0 int 父编号
CHNLORDER 序号 0 int
SCHEDULE 计划发布任务编号 int 保留,计划发布
CHNLDATAPATH 存放位置 nvarchar 200 保留,存放位置
CHNLORDERBY 栏目文档排序方式 nvarchar 200
ATTRIBUTE 扩展属性 nvarchar 500
CRUSER 对象创建者 nvarchar 50 WCMUSER.USERNAME
CRTIME 对象创建时间 datetime
STATUS 状态 0 int 状态
CHNLTYPE 类型 0 int
CHNLOUTLINETEMP 概览模板 0 int 保留
CHNLDETAILTEMP 细览模板 0 int
PUBLISHPRO 发布属性 1 smallint
OPERUSER 删除栏目的操作用户 nvarchar 50
OPERTIME 被删除的时间 datetime
LINKURL 链接地址 nvarchar 200 链接型栏目有效
CONTENTADDEDITPAGE 自定义编辑页面 nvarchar 200
CONTENTLISTPAGE 自定义内容(文档)列表页面 nvarchar 200
CONTENTSHOWPAGE 自定义内容查看页面 nvarchar 200
OUTLINEFILEDS --列说明-- nvarchar 1000
OUTLINEDBFIELDS --列说明-- nvarchar 1000

备注

  1. 名称：栏目名称在同一个站点下唯一
  2. 属性：是否允许发布,管理节点等属性
  3. 父编号：站点一级子栏目父编号为0
  4. 计划发布：计划任务
  5. 存放位置：见发布设置WCMFOLDERPUBLISHCONFIG.DATAPATH
  6. 状态：共三个有效值[正常=0,已删除=-1,父对象已删除=-2]
