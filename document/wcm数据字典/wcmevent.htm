WCMEVENT表：

描述：

系统的日程表事件

结构：

列名 说明 默认值 允许空值 类型 长度 备注
EVENTID 编号 int
OWNER 操作所有者 nvarchar 50
TITLE 事件标题 nvarchar 50
EDESC 事件说明 nvarchar 500
ETYPE 类型 smallint WCMEVENTTYPE.EVENTTYPEID
ESTART 开始时间 datetime
EEND 结束时间 datetime
EMAIL 电子邮件地址 nvarchar 1000
ATTRIBUTE 扩展属性 nvarchar 500
ISSHARE 是否共享 smallint
ONLINEREMIND 是否在线短消息提醒 smallint
EMAILREMIND 是否邮件提醒 smallint
MOBILEREMIND 是否手机短信提醒 smallint
AHEADREMDTIME 距离开始多少时间提醒 int
CRUSER 对象创建者 nvarchar 50 WCMUSER.USERNAME
CRTIME 对象创建时间 datetime
REMINDMSGID 提醒消息编号 0 int
